# MEANBoilerplate

A MEAN Full Stack boilerplate

## Boilerplate contents

* Angular 5 front-end
* Node.js back-end
* Express server
* MongoDB with Mongoose
* Bootstrap styling

## How to run

1. Clone the repo and navigate to the project folder using the terminal
2. run `npm install`
3. run `ng build`
4. run `node server`
5. navigate to `localhost:3000` on your browser
